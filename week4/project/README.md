# Project 4 : Arabic RAG system with Gradio interface

## Data set:
use 30000 arabic wikipedia samples from [Jr23xd23/ArabicText-Large](https://huggingface.co/datasets/Jr23xd23/ArabicText-Large).

## Embedding Model:
use [Omartificial-Intelligence-Space/Arabic-Triplet-Matryoshka-V2](https://huggingface.co/Omartificial-Intelligence-Space/Arabic-Triplet-Matryoshka-V2).

## chunk functions:
there are 3 type of chunking:
- fixed_chunking
- recursive_chunking
- semantic_chunking

## LLM Model:
used **llama-3.1-8b-instant** with **Groq**
